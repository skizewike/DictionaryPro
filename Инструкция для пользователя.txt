Инструкция для пользователя: Полный запуск DictionaryPro
Что вам прислали
Вам прислали архив с программой DictionaryPro для изучения иностранных слов. Распакуйте архив в любую удобную папку, например, на рабочий стол.

Шаг 1. Установка Python (делается один раз)
Откройте сайт: https://www.python.org/downloads/

Нажмите желтую кнопку "Download Python"

Запустите скачанный файл

В самом начале установки ОБЯЗАТЕЛЬНО отметьте галочку "Add Python to PATH"

Нажмите "Install Now"

Дождитесь окончания установки

Шаг 2. Запуск программы
Откройте папку DictionaryPro

Найдите файл start.bat

Запустите его двойным кликом мыши

Подождите 5-10 секунд — откроется окно браузера

Шаг 3. Регистрация
В открывшемся окне нажмите "Зарегистрироваться"

Придумайте логин (например, ваш никнейм)

Введите email (можно любой, даже выдуманный)

Придумайте пароль (запомните его!)

Нажмите "Создать аккаунт"

Вы автоматически войдете в программу

Шаг 4. Создание суперпользователя (для добавления слов)
Суперпользователь — это администратор, который может добавлять языки, уроки и слова. Создайте его один раз.

Закройте программу (закройте браузер и черное окно)

Откройте папку DictionaryPro

В адресной строке папки напишите cmd и нажмите Enter

В появившемся черном окне напишите команду:

text
python manage.py createsuperuser
Введите имя пользователя (например, admin)

Введите email (можно любой)

Введите пароль (запомните его!)

Подтвердите пароль

Если спросит "Bypass password validation", нажмите y и Enter

Готово! Суперпользователь создан.

Шаг 5. Добавление языков, уроков и слов
Снова запустите start.bat

В браузере перейдите по адресу: http://127.0.0.1:8000/admin

Введите логин и пароль суперпользователя (который создали в шаге 4)

Вы попадете в админ-панель

Добавление языка:
Нажмите "Добавить" рядом с "Языки"

Название: например, "Английский"

Код: например, "en"

Нажмите "Сохранить"

Добавление урока:
Нажмите "Добавить" рядом с "Уроки"

Название: например, "Урок 1. Еда"

Язык: выберите созданный язык

Описание: например, "Слова на тему еды"

Отметьте "Активен"

Нажмите "Сохранить"

Добавление слов:
Нажмите "Добавить" рядом с "Слова"

Иностранное слово: например, "apple"

Перевод: например, "яблоко"

Язык: выберите язык

Урок: выберите урок

Категории: можете пропустить

Нажмите "Сохранить"

Добавьте столько слов, сколько хотите выучить.

Шаг 6. Начало обучения
Нажмите на логотип "DictionaryPro" вверху или перейдите на http://127.0.0.1:8000

Нажмите "Изучение" в верхнем меню

Выберите созданный урок

Нажмите "Начать урок"

Вводите перевод слов и нажимайте "Проверить"

После прохождения урока нажмите "Сохранить результат"

Как закрыть программу
Закройте окно браузера

Закройте черное окно командной строки (нажмите на крестик)

Что делать, если что-то пошло не так
Ошибка "Python is not recognized" — Python не установлен. Вернитесь к шагу 1.

Ошибка "No module named 'django'" — откройте папку DictionaryPro, в адресной строке напишите cmd, в черном окне напишите pip install django и нажмите Enter.

Забыли пароль суперпользователя — создайте нового. Откройте cmd в папке DictionaryPro и напишите python manage.py createsuperuser.

Нет слов при запуске урока — вы не добавили слова в админке. Вернитесь к шагу 5.

Не могу зайти в админку — вы используете обычный аккаунт, а не суперпользователя. Войдите под логином и паролем, которые создали в шаге 4.

Порт 8000 занят — в адресной строке браузера введите http://127.0.0.1:8001

Приятного изучения! 📖
